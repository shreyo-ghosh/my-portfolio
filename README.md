## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/[repository-name].git
cd [repository-name]
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run deploy` - Deploy to GitHub Pages

## 🎨 Customization

### Updating Contact Information

Edit the contact details in `src/Home.jsx`:

```jsx
// Update email
<a href="mailto:your-email@example.com">

// Update phone
<a href="tel:+1234567890">

// Update social links
<a href="https://linkedin.com/in/yourprofile">
<a href="https://github.com/yourprofile">
```

### Modifying Services and Pricing

Update the `services` array in `src/Home.jsx` to modify:
- Service descriptions
- Pricing
- Package features
- Service icons

### Styling Changes

The portfolio uses custom CSS located in `src/index.css`. Key style sections:
- Utility classes for layout and spacing
- Custom component styles (`.service-card`, `.package-card`, `.price-tag`)
- Responsive breakpoints
- Color scheme and gradients

## 🚀 Deployment

### GitHub Pages (Automatic)

1. Push changes to your repository
2. Run the deploy command:
```bash
npm run deploy
```

The site will be automatically built and deployed to GitHub Pages.

### Manual Deployment

1. Build the project:
```bash
npm run build
```

2. Deploy the `dist/` folder to your hosting provider of choice.
